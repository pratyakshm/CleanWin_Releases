# Fixes
### This release fixes the following issues:
- Connect app not removing in Windows 1909 and below.
- Services not stopping for some users across reboots.
- OneDrive not uninstalling properly for some users.
- WinGet not installing properly for some users.
- Windows Search not working properly after running CleanWin.
# Changelog
## Apps & Features
- Add app "Microsoft.GamingServices" into the app removal list.
- Add Microsoft To Do into the app removal list.
- Add Power Automate Desktop into the app removal list.
- Enable Windows Sandbox and Windows Subsystem for Linux.
- [CLI] Enable Startup boost in Microsoft Edge (uncomment function to run).
- Replace Chocolatey Package Manager with Windows Package Manager.
- [CLI] Remove "Web Browsing - Restore recommended" suggestion in Settings using [ViVeTool](https://github.com/thebookisclosed/ViVe).
- [CLI] Cleanup duplicate code to cleanup registry.
- Cleanup duplicate code that prints messages.
- Check for Winstall.txt file and inform user if file not found.
- [CLI] Introduce seperate function for unpinning start menu tiles.

## General 
- [CLI] Display OS Build info
    - ProductName
    - ReleaseId
    - DisplayVersion
    - BuildLab
- [CLI] Create System Restore point of type "MODIFY_SETTINGS" if not created in the last 24 hours.
- [CLI] Make CleanWin run faster by removing most Sleep points.
- Create logs in C:\CleanWin instead of C:\Temp\CleanWin.
- [CLI] Exit Maintenance mode and re-enter back into regular servicing channel.
- [CLI] Make the run fully unattended
- CleanWin is now PowerShell 7 ready.
- Add individual scripts folder called "scripts" (I could not come up with a better name) in the root directory of CleanWin repository (main branch) for users that might be interested.

## Privacy & Security
- [CLI] Switch from unified functions to individual functions in Privacy section.
- Add new values to turn off sync provider notifications.
- Move "DiagTrack" and "DMWAppPushService" from Privacy & Security to Tasks & Services.
- No longer turn off speech recognition, because the code for it no longer works in newer editions of Windows (202H2 and up). A fix is being worked upon.
- [CLI]  No longer turn off Background apps by default because that tends to mess with Windows Search for some users.

## Tasks and Services
- Add "Setup Windows Update" and "Reset Windows Update".
    - What "Setup Windows Update" does:
        - Turn off automatic updates
        - Delay feature update by 20 days
        - Delay quality updates by 4 days
        - Turn off drivers offering through Windows Update
        - Turn off installation of bloatware following Windows Feature updates
    - "Reset Windows Update" reverts all settings back to the stock settings.
    
- [GUI] Unify "Disable unnecessary tasks" & "Disable unnecessary services" buttons.
- Print names of tasks and services that get disabled/enabled.


## Windows Explorer
- [GUI] Remove all individual changes buttons and unify them into a single one.
- [GUI] Add "Unpin tiles from Start menu" button.
- Cleanup duplicate code that unpins start menu tiles.
- No longer enable Verbose Status by default.
